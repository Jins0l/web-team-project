# web-team-project
# 1. 프로젝트 소개 및 설정 이유
이 프로젝트는 HTML, CSS를 사용하여 실제 운영 중인 서비스인 와챠피디아를 클론코딩 했습니다.
기본적인 웹 퍼블리싱을 연습하고 Git과 Github을 사용하여 협업 과정을 연습했습니다.

# 2. 사용기술
- HTML5
- CSS

# 3. 팀원 역할
| 이름  | 담당영역       |
|-----|------------|
| 정진솔 | 메인         |
| 홍의진 | 영화, 시리즈 |
| 육현승 | 웹툰, 책  |

# 4. 프로젝트 구조
```
┣ 📂assets
```
